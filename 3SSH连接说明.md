# SSH连接说明：
- cd openwrt && make menuconfig
#
- SSH连接说明：复制代码-->粘贴到putty开始连接-->等待跑码完成-->按Ctrl+c-->输入cd openwrt && make menuconfig然后回车-->等待跑码完成进入配置画面-->配置完成后-->一路Exit退出配置界面-->Ctrl+d结束SSH连接
#
- 必定要正确按键盘Ctrl+d结束SSH连接才会继续编译，不结束的话，或者打开SSH后没管的话，30分钟后编译失败
#
#
## 打开了SSH的话，一定要打开这个编译画面等待SSH展开，在‘启动教程’有介绍，如果到了这里，SSH展不开来就立马刷新下网页再等待一会看看，还是展不开来的话，只能关闭这个编译再重新开始了，那也是没办法的，有时候卡网页展不开也正常的，大家体谅下
#
